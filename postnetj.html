Internet Online POSTNET and PLANET Barcode Decoder
by Douglas Boynton Quine


POSTNET and PLANET DECODE software enables you to type in a United States Postal Service POSTNET or PLANET barcode using "1" and "." symbols and receive in response the decoded ZIP code or tracking ID. This JavaScript decoder supports and reports the extended PLANET code with 2 added digits. If the POSTNET or PLANET barcode is incomplete, contains a bad checksum, or contains bad digits, the program flags the errors. Decode USPS POSTNET (ZIP code)
and PLANET (tracking) barcodes.

  • Type the POSTNET or PLANET barcode into the box below using number "1" for tall bars and period "." for short bars (for example: 1..1.1..1.111.....1.1..1.1..1.11)
  • Press the "Decode Barcode" button when the barcode is entered.
  • If you make an error while typing the barcode, "backspace", and retype the incorrect characters.
  • POSTNET or PLANET barcode will be decoded.
  • POSTNET or PLANET barcode type will be reported (types described below)
  • Partial barcodes, checksum errors, digit errors, missing start or end bars will be reported.
  • Find POSTNET barcode specification in the USPS DMM C840 POSTNET barcoding standards for letters and flats
  • Find ZIP code for a city at: USPS City State / ZIP Code Associations
  • Find city name for a ZIP code at: USPS All Cities/Towns in a ZIP code
  • Decode USPS POSTNET / PLANET Barcode:

    POSTNET or PLANET barcode types are:

        POSTNET (ZIP code)
       A  Field"   5 digit ZIP (e.g. 12345)
       C  Field"   9 digit ZIP (e.g. 12345-6789)
       C' Field"  11 digit ZIP (e.g. 12345-6789-01)
       B  Field" adds 6 digits (e.g.    45-6789)   (makes 9 digit ZIP)
       B' Field" adds 8 digits (e.g.    45-6789-01)(makes 11 digit ZIP)
    PLANET (tracking barcode)
       origin confirm barcode with customer ID (e.g. 21-123456789-2)
             service 21 = residual mail
             service 50 = courtesy reply letters
             service 51 = courtesy reply flats
             service 52 = business reply letters
             service 53 = business reply flats
             service 54 = business reply cards
             service 56 = QBRM letters
             service 57 = QBRM cards
             service 58 = courtesy reply cards
       destination confirm barcode with subscriber and mailing ID (e.g. 22-12345-6789-2)
             service 22 = residual mail
             service 30 = ? computerized forwarding system code for forwarded mail (followed by destination ZIP code) one observation by Terry Hines in Jan. 2013 - please report any others
             service 40 = first class letters
             service 41 = first class flats
             service 42 = standard letters
             service 43 = standard flats
             service 44 = periodical letters
             service 45 = periodical flats
             service 46 = first class cards
             service 47 = standard cards

FAQ - Frequently Asked Questions

  • POSTNET barcodes are overwhelming, can you help me?

    barcode404.html

  • Will the POSTNET DECODE program decode the new four state (full/descending/ascending/track) USPS Intelligent Mail Barcodes (IMB)?

    No, however the USPS online IMB decoder can do so.

  • I am looking for a POSTNET bar code font to barcode my mail and obtain postal discounts

    The WORD word processor from MICROSOFT (check under "Envelopes" options) and many other programs (such as: My Advanced Mailing List available at Office Superstores and Postalsoft "Address Dictionary" from 1-800-831-6245) will create POSTNET barcodes to go with addresses on envelopes. Such programs, however, do not meet United States Postal Service (USPS) CASS certification regulations unless the ZIP codes have been confirmed against a licensed database within the last 6 months. Some vendors offer subscription services to provide the mandatory regular updates to maintain certification. In short getting the font to print the barcodes can be achieved through several routes but the USPS won't give you a discount. Naturally the postal service's concern is that wrong barcodes are worse than none because they misdirect the mail. Perhaps the United States Postal Service publication Mailer's Companion shows this question best. goodbar.gif

  • What software is USPS CASS certified to barcode my mail and obtain postal discounts?

    1/ PITNEY BOWES Shipping & Weighing: "SmartMailer" through 1-800-MR-BOWES (needs annual subscription)

    2/ POSTALSOFT: "DeskTop Postal Code" through 800-831-6245 (needs annual subscription)

    3/ MySoftware Co: "My Advanced Mailing List" through office superstores (needs annual subscription)

  • National Address Server: http://www.cedar.buffalo.edu/adserv.html

    Given a valid U.S. postal address, this server attempts to rewrite the address in the proper format along with the ZIP+4 code. If it is successful, you can retrieve a Postscript or a GIF file of the address for printing, with a barcode! You can also view a street map of the address, from 2 different Internet map sites (MapBlast and MapQuest).

  • What Does the USPS Charge for PLANET/CONFIRM Code Tracking / Tracing?

    The PLANET code program has been operated as a no charge pilot program but it is moving towards a paid service. The "POSTCOM Bulletin" (April 26, 2002 page 6) reports the following three tier pricing structure was filed as a rate case by the USPS on April 24, 2002. (The average mail piece is scanned 3 times on the journey through the USPS.)

    1. Silver: 3 month subscription, 1 mailing ID code, 15 million scans, $2,000 (additional ID mailing codes are $500 each; additional 2 million scans are $500)
    2. Gold: 1 year subscription, 1 mailing ID code, 50 million scans, $4,500 (additional ID mailing codes are $2,000 each; additional 6 million scans are $750)
    3. Platinum: 1 year subscription, 3 mailing ID codes, unlimited scans, $10,000 (additional ID mailing codes are $2,000 each)
